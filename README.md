# 日本 観光来訪者数マップ

日本全国の観光来訪者数を、都道府県レベルでインタラクティブに可視化するウェブアプリです。

---

## 機能

### 地図（全国ビュー）

- **コロプレスマップ**：来訪者数の多い都道府県ほど濃い赤で色分け
- **沖縄インセット**：天気予報形式で左上に独立表示
- **表示モード切替**：来訪者数の絶対数 ↔ 人口あたりの比率
- **年月選択**：2021年1月〜2026年3月を任意に選択
- **前年比 / 前月比**：差を青（減少）〜グレー〜赤（増加）の発散型カラーで表示
- **ホバーツールチップ**：都道府県名・来訪者数・人口比率・変化率を表示

### グラフ（都道府県詳細ビュー）

- 都道府県をクリックすると、**2021年〜2026年3月の来訪者数推移**を縦棒グラフで表示
- 月単位（63本）と年単位（8本）を切り替え可能
- 来訪者数と人口比率を切り替え可能
- バーの色は年ごとに変化（古い年ほど薄い赤 → 新しい年ほど濃い赤）

## データ出典

| データ | 出典 | 期間 |
|---|---|---|
| 観光来訪者数 | [デジタル観光統計オープンデータ](https://www.nihon-kankou.or.jp/home/jigyou/research/d-toukei/)（日本観光振興協会） | 2021年〜2026年3月 |
| 都道府県別人口 | [総務省 住民基本台帳人口](https://www.soumu.go.jp/main_sosiki/jichi_gyousei/daityo/jinkou_jinkoudoutai-setaisuu.html) | 2021〜2026年（各年1月1日現在） |
| 都道府県境界 | [dataofjapan/land](https://github.com/dataofjapan/land) | - |

## ローカルで動かす

```bash
git clone https://github.com/ItalyFuji/japan_tourist-visitors_map.git
cd japan_tourist-visitors_map

# データ変換（初回のみ）
pip install openpyxl
python scripts/convert_data.py

# ローカルサーバー起動
python -m http.server 8080
```

ブラウザで `http://localhost:8080` を開く。

> HTMLファイルをダブルクリックで開くとCSVが読み込めないため、ローカルサーバーが必要です。

## 来訪者数の予測について

将来の来訪者数予測は **技術的には実装可能**ですが、現バージョンでは実装していません。

理由：観光データは自然災害・感染症・政府の観光施策・為替・大型イベントといった**外部要因の影響が非常に大きく**、数値モデルが織り込みにくいため。また、手元のデータは2021〜2026年の約5年分と短く、COVID-19の影響で2021〜2022年のデータが歪んでいることも精度低下の要因となります。

詳細は [SPEC.md §9](SPEC.md#9-来訪者数予測について) を参照。
